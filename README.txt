**Summary:**
We wanted to create a program that would train itself to recognize numbers and work with handwritten images that we uploaded. We used Tensorflow on an environment created on Anaconda and run on the Spyder Python IDE in Anaconda using Python. This code works perfectly and recognises handwritten and printed numbers. We tested our product by pulling a random image from the internet and testing it using our program. We used code provided to us from resource 4 to unzip and stack the mnist files.
